# Heart Disease Screening Streamlit App

## Files
- `app.py`
- `requirements.txt`
- `heart_disease_health_indicators.csv` (place beside `app.py` if you do not upload it in the app)

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## What the app includes
- Screening input panel
- Threshold-optimized prediction
- Model performance dashboard
- SHAP global and local interpretation
- Dataset audit table

## Important
This is a screening app, not a diagnostic medical device.
